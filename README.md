# 📝 Cloud Notepad

A simple yet powerful **real-time cloud notepad** built with Firebase.
Write, paste, or upload text files — and access them from anywhere 🌍

---

## 🚀 Live Features

✨ Real-time sync (like Google Docs)
💾 Auto-save while typing
📂 Upload `.txt`, `.html`, `.js`, `.php` files
📋 One-click copy to clipboard
🔄 Reset for all users
⚡ Fast & lightweight (pure HTML + JS)
🌐 Works on GitHub Pages

## ⚙️ Setup Guide

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/notepad.git
```

### 2️⃣ Open Project

Just open `index.html` in browser OR deploy to GitHub Pages.

---

### 3️⃣ Firebase Setup

* Go to Firebase Console
* Create project
* Enable **Firestore Database (Test Mode)**

Paste config in your code:

```js
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "...",
  projectId: "..."
};
```

## 📋 How It Works

* Typing → Auto saved to cloud
* Refresh → Data remains
* Multiple users → Same content visible
* Upload file → Loads content into editor
